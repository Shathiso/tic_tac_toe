# Tic Tac Toe

## Project setup

### Switch Node version
```
nvm use 16
```

### Install npm packages
```
npm install
```

## Run tests
```
npm run test
```

## Run a local dev server
```
npm run dev
```

## Access the application
http://localhost:5173/ or view it on Github pages at :https://shathiso.github.io/tic_tac_toe/
